Project Description

This project showcases a deep dive into competitive intelligence, specifically targeting academies in the Data Analysis sector. A robust data model was engineered using Excel for data collection (social media stats, pricing tables) and the visualization was executed in Power BI.

The final interactive dashboard provides immediate insights: a clear pricing map identifying high-value and low-cost competitors, a digital performance scorecard revealing which academies drive the most engagement, and a market share breakdown based on follower base (e.g., VAT and IBS are the market leaders).

This project demonstrates strong skills in data modeling, competitive analysis, and strategic dashboard design that informs pricing, marketing, and business decisions.

  • Developed a fully interactive Power BI dashboard with dynamic filters
  • Applied calculated measures for engagement rate and price benchmarking
  • Used DAX functions for KPI standardization across multiple datasets
  • Implemented design best practices for visual storytelling and stakeholder impact
