Sales Performance Dashboard
Using Tableau Public

A comprehensive sales analytics dashboard built in Tableau Public featuring 15+ interactive visualizations including KPI metrics, trend analysis, and geographic insights. Designed for stakeholders to track business performance across categories, regions, and time periods with clean formatting and intuitive navigation.

Project Overview

This project demonstrates end-to-end dashboard development using Tableau Public, transforming raw sales data into actionable business intelligence. The dashboard includes five key performance indicators (Total Sales, Orders, Profit, Customers, and Products Sold) displayed as formatted BANs, complemented by monthly trend line charts for each metric.

The visualization suite features three horizontal bar charts analyzing sales by sub-category, category, and region with custom highlighting for top performers. A pie chart breaks down sales by customer segment, while an interactive map provides geographic distribution insights across states.

What I Built

  • 5 KPI BANs: Total Sales, Orders, Profit, Customers, and Products Sold with custom formatting and number styling
  • 5 Line Charts: Monthly trend analysis for each KPI with zero line emphasis and clean tooltips
  • 3 Bar Charts: Sales breakdown by Sub-category, Category, and Region with top performer highlighting
  • 1 Pie Chart: Sales by Customer Segment with color gradients based on performance
  • 1 Map Chart: Geographic sales distribution across states with color-coded volume indicators

Key Skills Demonstrated

  • Data connection and preparation using Tableau Public Desktop
  • Mastered Dimensions vs Measures and Continuous vs Discrete field concepts
  • Created calculated fields for conditional formatting and top performer identification
  • Applied dashboard composition best practices using containers, padding, and strategic whitespace
  • Implemented stakeholder-ready formatting with cleaned tooltips and appropriate number scales (K/M)
  • Published interactive dashboard to Tableau Public for sharing and collaboration
